# 3DEngine

A 3D Java game engine 
(Screenshots below)

## Features

1. Terrain Generation
2. Shadow
3. Particle System
4. Normal Maps
5. 3D Model Support

## Getting Started

* Clone project and run via...
* src > engineTester > MainGameLoop.java

## Built With

* Java
